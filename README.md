<h1 align="center">SetUp Windows Subsystem for Linux</h1>

#### Run the following command to configure the terminal with `zsh` and `oh my zsh`

```shell
# cloning repository
git clone https://github.com/Armando-Rios/Setup.git

# enter Setup
cd Setup

# running script
bash setup.sh
```
